## 1 排序
平均，最好，最坏，空间复杂度，是否稳定
* 插入排序 O(n^2)   O(n^2)   O(n)     O(1)    稳定
* 快速排序 O(nlogn) O(n^2)   O(nlogn) O(logn) 不稳定
* 堆排序     O(nlogn) O(nlogn) O(nlogn) O(1)    不稳定
* 归并排序 O(nlogn) O(nlogn) O(nlogn) O(n)    稳定
